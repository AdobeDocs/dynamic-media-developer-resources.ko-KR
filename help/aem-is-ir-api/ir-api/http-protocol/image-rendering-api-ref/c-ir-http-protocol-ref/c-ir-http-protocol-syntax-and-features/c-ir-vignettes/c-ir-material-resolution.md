# 재질 해상도{#material-resolution}

반복 가능한 텍스처 또는 다른 재료의 해상도 값은 장면 좌표 공간에서 재료 이미지의 크기를 정의합니다.

텍스처 해상도 값은 인치당 픽셀(또는 비네팅이 장면 좌표 공간에 대해 지정하는 단위)로 지정됩니다.

재질 확인이 `res=`(으)로 지정되었습니다.
