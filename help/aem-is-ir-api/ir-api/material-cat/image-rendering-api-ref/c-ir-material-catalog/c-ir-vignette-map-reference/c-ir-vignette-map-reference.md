# 비네팅 맵 참조{#vignette-map-reference}

재질 카탈로그에 비네팅 맵이 필요한 경우, 비네팅 맵 파일의 상대 경로나 절대 경로를 attribute::VignetteMapFile에 지정해야 합니다.

특정 재질 카탈로그의 비네팅 맵 파일 항목은 기본 카탈로그의 비네팅 맵 파일 항목을 재정의합니다.
