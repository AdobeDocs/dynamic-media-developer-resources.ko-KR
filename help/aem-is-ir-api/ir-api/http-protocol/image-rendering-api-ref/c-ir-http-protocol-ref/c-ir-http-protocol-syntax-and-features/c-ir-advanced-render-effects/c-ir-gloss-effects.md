# 광택 효과{#gloss-effects}

비네팅에서 활성화된 경우 렌더러는 조명 알고리즘을 변경하여 재료 광택의 변화를 시뮬레이션합니다(주로 대비를 제어하여).

이 효과는 반사 렌더링과 무관하며, 반사 반사를 시뮬레이트하여 보완합니다. 반사를 제어하는 데 사용되는 동일한 `gloss=`, `glossmap=` 및 `type=` 특성으로 효과가 제어됩니다.
