# FontPath{#fontpath}

글꼴 파일 경로. 파일 접미사를 포함하여 주 글꼴 파일의 경로 및 이름입니다.

## 속성 {#section-ae2b7daf74a4419ebcec8f13aa8c1462}

텍스트 문자열. 필수. 절대 경로나 상대 경로인 유효한 이미지 서버 파일 경로가 있어야 `attribute::RootPath`합니다.

## 기본값 {#section-3fb42f362de84fdb8ba9fe4cc9628aa3}

없음.
