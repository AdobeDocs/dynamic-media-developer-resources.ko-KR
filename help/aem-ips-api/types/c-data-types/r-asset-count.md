# 자산 수{#assetcount}

회사의 자산 유형 및 카운트입니다.

자산 유형은 [!DNL ViewerSWf], [!DNL PDF], [!DNL Cabinet] 등을 포함합니다.

| 이름 | 유형 | 설명 |
|---|---|---|
| `*`assetType`*` | `xsd:string` | 개수와 연결된 자산 유형입니다. |
| `*`계수`*` | `xsd:int` | 회사의 `assetType` 자산 수입니다. |
