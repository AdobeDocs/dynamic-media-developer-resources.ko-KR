## 속성 {#section-e8449b2225294518a5f9ddb698f68360}

텍스트 문자열 값입니다. 카탈로그 폴더에 상대적인 절대 파일 경로 또는 경로입니다.

## 기본값 {#section-20a9d535d64347abaf0233595b2ad089}

비어 있음 - 이 이미지 카탈로그에 비공개 요청 사전 처리 규칙이 없음을 나타냅니다.
