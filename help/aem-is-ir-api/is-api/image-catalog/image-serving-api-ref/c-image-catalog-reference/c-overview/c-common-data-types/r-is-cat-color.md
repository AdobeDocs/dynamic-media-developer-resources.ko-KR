# 컬러{#color}

색상 데이터 유형.

색상 값. 16진수로 채워진 RGB 값(선택 사항: `0x` 포함) 예를 들어 RGB 값 `128,255,0`은 `0x80ff00` 또는 `80ff00`로 지정할 수 있습니다.
