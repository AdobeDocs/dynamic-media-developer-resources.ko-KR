# ApplyEffectMask{#applyeffectmask}

효과 레이어에 최종 마스크를 적용하여 효과와 레이어 간에 겹치는 영역을 마스크할 수도 있습니다.

유효한 값은 0과 1입니다. 0은 효과 마스크를 적용하지 않음을 의미합니다. 1은 효과 마스크를 적용함을 의미합니다. 기본값은 1입니다.
