# RootId{#rootid}

카탈로그 식별자. 요청의 이미지 개체 지정자에서 이 카탈로그를 식별하는 데 사용되는 HTTP 경로 요소입니다.

## 속성 {#section-9a49da71de634378a06d2347790898a0}

텍스트 문자열 값입니다. HTTP 경로에서 유효한 문자만 포함할 수 있습니다.

## 기본값 {#section-c5296f4e52394984bf1c0d265ecde940}

없음. 각 카탈로그에는 고유한 `attribute::RootId` 값이 있어야 합니다. [!DNL default.ini]에는 일반적으로 빈 `attribute::RootId`이(가) 있습니다.
