검색 결과에서 제외할 생성 엔진 및 생성된 자산 유형을 결정합니다.

|---|---|---|
| ` *`엔진`*` | `xsd:string` | 제외할 자산을 만든 생성 엔진입니다. 값은 생성 정보를 참조하십시오. |
