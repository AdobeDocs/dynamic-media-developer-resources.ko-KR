# 라이선스{#licensing}

Dynamic Media 이미지 렌더링 라이선스 키가 필요하며 이미지 렌더링이 설치된 각 시스템에 설치해야 합니다.

이미지 렌더링은 유효한 라이선스 없이 실행되지 않습니다. 설치 절차에는 라이센스 키를 가져오고 설치하는 단계가 포함됩니다.
