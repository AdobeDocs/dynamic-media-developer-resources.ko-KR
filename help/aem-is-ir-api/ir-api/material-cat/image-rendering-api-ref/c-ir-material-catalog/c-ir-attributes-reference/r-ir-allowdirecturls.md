# AllowDirectURL{#allowdirecturls}

절대 소스 URL 허용

명령에서 절대 HTTP URL(&quot; `http://`&quot;로 시작)을 허용하려면 1로 `src=` 설정합니다. 절대 URL을 허용하지 않으려면 0으로 설정합니다.

## 속성 {#section-9246bc06c7484f9e8f65a5edb24f6854}

플래그.

## 기본값 {#section-a881c30ea28e47e296faa890c3bd6daf}

정의되지 않았거나 비어 있는 `default::AllowDirectUrls` 경우 상속됩니다.
