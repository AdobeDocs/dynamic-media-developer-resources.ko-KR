# AssetCount{#assetcount}

회사의 자산 유형 및 카운트.

자산 유형에는 [!DNL ViewerSWf], [!DNL PDF], [!DNL Cabinet] 등이 있습니다.

| 이름 | 유형 | 설명 |
|---|---|---|
| `*`assetType`*` | `xsd:string` | 계수와 연관된 자산의 유형입니다. |
| `*`계수`*` | `xsd:int` | 회사의 `assetType` 자산 수입니다. |
