# ActiveJob{#activejob}

서버에서 실행되는 작업입니다. 또한 예약된 작업의 인스턴스입니다.

작업은 3개 주에 있습니다.

* 실행되도록 예약되었습니다.
* 현재 실행 중입니다.
* 실행 완료(및 작업 로그에 이미 정보를 기록함).

작업 유형을 반환할 작업 유형 값을 지정합니다. 다음 작업을 반환할 수 있습니다.

* `imageServingPublishJob`
* `imageRenderingPublishJob`
* `videoPublishJob`
* `serverDirectoryPublish`
* `JobUploadDirectoryJob`
* `uploadUrlsJob`
