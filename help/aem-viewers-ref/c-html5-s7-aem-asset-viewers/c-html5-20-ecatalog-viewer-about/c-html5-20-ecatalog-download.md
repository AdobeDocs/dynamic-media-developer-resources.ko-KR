# 다운로드{#download}

제어 막대의 &quot;다운로드&quot; 단추를 사용하여 전자 카탈로그를 PDF 파일로 다운로드할 수 있습니다.

다음과 같은 경우 뷰어 사용자 인터페이스에서 &quot;다운로드&quot; 단추를 자동으로 사용할 수 있습니다.

* 실제 PDF 파일이 고객 회사에 있습니다.
* 이 이름은 뷰어에 전달되는 전자 카탈로그 에셋의 이름과 일치하며 `.pdf` 확장명을 포함합니다.
* PDF 파일이 SPS에 게시됩니다.
