# 텍스트 문자열{#text-string}

텍스트 문자열 데이터 유형입니다.

문자열에 `<CR>`, `<LF>` 또는 `<TAB>` 문자가 포함되어 있지 않으면 문자열 구분 기호는 선택 사항입니다. 단일 또는 큰따옴표를 구분 기호로 사용할 수 있습니다. 인용 부호를 사용하는 경우 문자열 내에 포함된 이러한 인용 부호는 연속적인 두 따옴표(예:이번 달 특별).
