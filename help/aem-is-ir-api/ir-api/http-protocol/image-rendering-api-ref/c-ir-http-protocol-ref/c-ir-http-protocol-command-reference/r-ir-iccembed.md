# iccEmbed{#iccembed}

ICC 프로파일 포함을 참조하십시오. 작업 중인 ICC 색상 프로파일 또는 icc=로 지정된 프로파일을 응답 이미지에 포함할지 여부를 지정합니다.

`iccEmbed=0|1`

## 속성 {#section-45c08073626a4cc7823d423dc35b4479}

요청 내 어느 곳에서든 발생할 수 있습니다.

## 기본값 {#section-68ed1f0c68dc46b58972804799c277b7}

`iccEmbed=0`를 선택합니다.
