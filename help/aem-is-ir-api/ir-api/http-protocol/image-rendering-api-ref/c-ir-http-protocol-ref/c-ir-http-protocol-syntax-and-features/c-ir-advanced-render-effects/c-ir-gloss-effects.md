# 광택 효과{#gloss-effects}

렌더러는 비네트에서 활성화된 경우 조명 알고리즘을 변경하여 재료 광택의 변형(주로 대비 제어)을 시뮬레이션합니다.

이 효과는 반사 렌더링과 독립적이며 분광 반사를 시뮬레이션하여 보완합니다. 이 효과는 반사를 제어하는 데 사용되는 동일한 `gloss=`, `glossmap=` 및 `type=` 속성으로 제어됩니다.
