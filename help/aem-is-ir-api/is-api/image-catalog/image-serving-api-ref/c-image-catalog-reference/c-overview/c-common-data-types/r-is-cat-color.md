# 컬러{#color}

색상 데이터 유형.

색상 값. 16진수로 채워진 RGB 값(선택 사항 뒤에 포함) `0x`입니다. 예를 들어 RGB 값은 `128,255,0` 또는 `0x80ff00` `80ff00` 으로 지정할 수 있습니다.
