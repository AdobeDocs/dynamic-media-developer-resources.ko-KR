# 광택 효과{#gloss-effects}

비네팅에서 활성화된 경우 렌더러는 조명 알고리즘을 다양하게 변경하여 재료 매끄러운 광택을 시뮬레이션합니다(주로 대비를 제어함으로써).

이 효과는 반사 렌더링과 무관하며 반사광 반사를 시뮬레이션하여 이를 보완합니다. 이 효과는 반사를 제어하는 데 사용되는 동일한 `gloss=`, `glossmap=` 및 `type=` 속성으로 제어됩니다.
