# IccBlackPointCompensation{#iccblackpointcompensation}

블랙포인트 보상. icc=로 명시적인 선택이 없을 때 색상 변환에 검은 점 보정을 적용할지 여부를 지정합니다.

## 속성 {#section-ea27b8089b89468bbc38e9e7154ea413}

플래그. 비활성화하려면 0으로, 블랙포인트 보정을 활성화하려면 1로 설정합니다.

## 기본값 {#section-0d79b203be4c434f927b7c03c7a0062d}

정의되지 않았거나 비어 있는 `default::IccBlackPointCompensation` 경우 상속됩니다.
