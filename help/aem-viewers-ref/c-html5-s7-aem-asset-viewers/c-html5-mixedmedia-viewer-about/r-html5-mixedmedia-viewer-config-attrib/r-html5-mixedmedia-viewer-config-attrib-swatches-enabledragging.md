# 색상 견본.활성화 드래그{#swatches-enabledragging}

` [Swatches.|<containerId>_swatches.]enabledragging=0|1[, *`overdragvalue`*]`

<table id="table_B1363BFD20204093AAB326A1AB503B93"> 
 <tbody> 
  <tr> 
   <td> <p> <span class="codeph"> 0 | 1 </span> </p> </td> 
   <td> <p> 사용자가 마우스로 견본을 스크롤하거나 터치 제스처를 사용하여 견본을 스크롤할 수 있도록 하거나 비활성화할 수 있습니다 </p> </td> 
  </tr> 
  <tr> 
   <td> <p> <span class="codeph"> <span class="varname"> overdragvalue </span></span> </p> </td> 
   <td> <p> 0- <span class="codeph"> 1 </span> 범위 내의 함수입니다. 실제 속도의 잘못된 방향으로 이동하는 <span class="codeph"> % </span> 값입니다. 이 값이 <span class="codeph"> 1로 </span>설정되면 마우스로 이동합니다. 0으로 설정된 경우 <span class="codeph"> 다른 </span>방향으로 전혀 이동할 수 없습니다. </p> </td> 
  </tr> 
 </tbody> 
</table>

선택 사항입니다.

`1,0.5`
