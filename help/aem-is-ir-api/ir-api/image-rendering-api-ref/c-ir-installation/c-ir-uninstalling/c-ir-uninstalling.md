# 이미지 렌더링 제거{#uninstalling-image-rendering}

필요한 경우 이미지 렌더링을 제거할 수 있습니다.

제거 시 설치 폴더에 있는 일부 사용자 문서가 제거될 수 있습니다. 제거하기 전에 유지할 모든 파일을 백업합니다.

제거하기 전에 관리자 또는 루트 권한으로 서버 호스트에 로그인합니다.
