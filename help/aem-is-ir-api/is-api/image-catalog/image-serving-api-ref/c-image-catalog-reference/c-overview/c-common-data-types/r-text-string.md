# 텍스트 문자열{#text-string}

텍스트 문자열 데이터 유형입니다.

문자열에 `<CR>`, `<LF>` 또는 `<TAB>` 문자가 포함되지 않는 한 문자열 구분 기호는 선택 사항입니다. 작은따옴표나 큰따옴표는 구분 기호로 사용할 수 있습니다. 따옴표를 사용하는 경우, 문자열 내에 포함된 이러한 따옴표는 두 개의 연속된 따옴표(예: 이번 달 특별 광고).
