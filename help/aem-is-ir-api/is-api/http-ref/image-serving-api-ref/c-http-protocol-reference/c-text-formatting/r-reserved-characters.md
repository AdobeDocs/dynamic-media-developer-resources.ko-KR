# 예약된 문자{#reserved-characters}

이 섹션에는 특정 용도로 예약된 문자가 나열됩니다.

RTF에서는 중괄호 &#39; `{`&#39; 및 &#39; `}`&#39;를 그룹 구분 기호로 사용합니다. 이 매개 변수는 쌍으로 발생해야 하며 중첩될 수 있습니다. 텍스트 문자열에 중괄호를 표시하려면 각각 &#39; `\{`&#39; 및 &#39; `\}`&#39;를 사용하십시오.

>[!NOTE]
>
>RTF에 사용된 모든 중괄호를 URL로 인코딩해야 합니다.

백슬래시 &#39;\&#39;는 RTF 명령 및 키워드를 적용하는 데 사용됩니다. 백슬래시를 표시하려면 `'\\'` 을 사용하십시오.
