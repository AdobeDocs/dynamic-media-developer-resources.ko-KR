# imageRes{#imageres}

Web to Print 요청에 포함된 비트맵의 해상도를 설정합니다.

` imageRes=< *`수`*>`

인쇄하려면 300을 권장합니다. 숫자가 클수록 이미지를 렌더링하는 데 더 많은 시간이 필요합니다.
