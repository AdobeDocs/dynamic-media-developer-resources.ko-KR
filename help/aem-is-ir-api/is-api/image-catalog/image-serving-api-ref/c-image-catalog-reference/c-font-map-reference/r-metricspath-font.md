# MetricsPath{#metricspath}

글꼴 지표 파일 경로입니다. 파일 접미사를 포함한 글꼴 지표 파일의 경로 및 이름입니다.

Adobe Type 1 글꼴에 사용됩니다. 지정하지 않을 경우, 서버는 주 글꼴 파일이 있는 동일한 폴더에서 글꼴 지표 파일을 찾으려고 시도합니다. 렌더링할 때 필요한 글꼴 지표 파일을 찾을 수 없는 경우 오류가 발생합니다.

## 속성 {#section-955268c581574875b05253d9e14544f3}

텍스트 문자열. Adobe Type 1 파일(선택 사항) 비어 있거나 올바른 이미지 서버 파일 경로(절대 또는 상대)여야 `attribute::RootPath`합니다.

## 기본값 {#section-a6ffbd6879c642caa5a2fd4ed14a3a85}

없음.
