# 색상{#color}

색상 데이터 유형.

색상 값. 16진수, 압축된 RGB 값(선택 사항 앞에 다음 포함) `0x`. 예를 들어 RGB 값 `128,255,0` 을(를) 다음으로 지정할 수 있습니다. `0x80ff00` 또는 `80ff00` .
