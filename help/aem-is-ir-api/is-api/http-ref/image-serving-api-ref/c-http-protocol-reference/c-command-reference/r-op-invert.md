# op_invert{#op-invert}

색상을 반전. 음수 이미지 효과의 각 색상 구성 요소를 반전합니다.

`op_invert=0|1`

## 속성 {#section-2e7a7b340dd147db9bd68ff3bfb6f4c1}

레이어 명령. `layer=comp`인 경우 현재 레이어나 합성 이미지에 적용됩니다. 효과 레이어에서 무시됩니다.

## 기본값 {#section-8040bc4e2a684c629a28b155aa0ba289}

`op_invert=0`을 클릭합니다.
