# loadcache{#loadcache}

서버 캐시를 미리 로드합니다. 요청을 req=img처럼 실행하지만 이미지를 반환하는 대신 서버는 MIME 유형 텍스트/plain의 텍스트 데이터로 형식이 지정된 응답 이미지(image.length)의 길이를 반환합니다.

`req=loadcache`

HTTP 응답을 액세스할 수 없습니다.

요청의 다른 명령은 문서화된 대로 적용됩니다.
