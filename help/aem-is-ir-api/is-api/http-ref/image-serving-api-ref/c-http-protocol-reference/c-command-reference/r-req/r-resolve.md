# 해결{#resolve}

요청 디버깅을 참조하십시오. 이 디버그 명령은 요청을 구문 분석하여 사전 처리하고, 이미지 카탈로그 조회, catalog::Modifier inclusions, macro 및 변수 대체 등을 req=img와 같이 실행합니다.

`req=resolve`

MIME 유형이 있는 결과 이미지 대신 최종 요청 문자열이 반환됩니다 `text/plain`.

HTTP 응답에 액세스할 수 없습니다.
