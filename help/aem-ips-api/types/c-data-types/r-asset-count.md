# 자산 수{#assetcount}

회사의 자산 유형 및 카운트.

자산 유형에는 [!DNL ViewerSWf], [!DNL PDF], [!DNL Cabinet]등이 포함됩니다.

| 이름 | 유형 | 설명 |
|---|---|---|
| ` *`assetType`*` | `xsd:string` | 카운트와 연결된 자산의 유형입니다. |
| ` *`계수`*` | `xsd:int` | 회사의 자산 `assetType` 수입니다. |
