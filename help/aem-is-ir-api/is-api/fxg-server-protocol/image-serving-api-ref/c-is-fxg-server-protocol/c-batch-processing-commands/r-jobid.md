# jobid{#jobid}

일괄 처리 작업의 작업 ID입니다.

상태 또는 출력이 요청된 작업을 지정하거나 특정 작업을 삭제하기 위해 배치 작업 요청에 사용됩니다. 값은 작업이 제출될 때 반환된 작업 ID 문자열입니다.
