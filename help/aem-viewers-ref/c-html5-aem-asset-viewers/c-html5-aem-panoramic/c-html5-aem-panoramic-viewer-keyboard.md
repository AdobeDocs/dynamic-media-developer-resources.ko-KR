# 키보드 접근성 및 탐색{#keyboard-accessibility-and-navigation}

뷰어 사용자 인터페이스에 노출된 모든 기능을 키보드로 액세스할 수 있습니다.
최종 사용자는 `Left`, `Right`, `UP` 및 `DOWN` 키 입력을 사용하여 이미지를 이동할 수 있습니다.
`Esc` 키 입력을 사용하여 이미지를 재설정할 수 있습니다.
