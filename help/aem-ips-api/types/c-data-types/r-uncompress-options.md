# UncompressOptions{#uncompressoptions}

ZIP 및 TAR 파일을 기본 자산(없음)으로 처리하거나 컨텐츠를 추출하고 업로드(압축 해제)하도록 설정을 업로드하십시오.

>[!NOTE]
>
>`None` 기본값은 기본값입니다.
