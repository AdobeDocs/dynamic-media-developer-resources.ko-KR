## 수정한 작업 {#section-596ea55a371e4c2ab5531e21ea9d8090}

**searchAsset**

* `name` 매개 변수를 제거했습니다.
* `excludeFieldArray`이(가) 추가되었습니다.

**getFolders**

* `excludeFieldArray`이(가) 추가되었습니다.

**getFolderTree**

* `excludeFieldArray` 및 `getUniqueMetadataValues`이(가) 추가되었습니다.
* `fieldHandle`이(가) 필수 매개 변수로 만들어졌습니다.
