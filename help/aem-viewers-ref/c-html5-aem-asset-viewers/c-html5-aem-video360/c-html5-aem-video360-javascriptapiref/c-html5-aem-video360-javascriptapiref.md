# Video360 뷰어에 대한 JavaScript API 참조{#javascript-api-reference-for-video-viewer}

다음 모든 예에서 `<instance>`은(는) `s7viewers.Video360Viewer` 클래스에서 인스턴스화된 JavaScript 뷰어 개체의 실제 이름을 나타냅니다.
