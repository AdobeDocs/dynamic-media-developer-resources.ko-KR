# setLocalizedText{#setlocalizedtexts}

슬라이드 뷰어에 대한 JavaScript API 참조.

` setLocalizedTexts( *`localizationInfo`*)`

하나 이상의 로케일에 대한 지역화 SYMBOL 값을 설정합니다. 이 매개 변수는 `init()` 전에 호출해야 합니다.
