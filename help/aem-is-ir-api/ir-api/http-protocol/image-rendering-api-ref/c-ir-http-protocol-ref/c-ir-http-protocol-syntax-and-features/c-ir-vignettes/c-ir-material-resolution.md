# 질감 해상도{#material-resolution}

반복 가능한 텍스처 또는 기타 재질의 해상도 값은 장면 좌표 공간의 재질 이미지의 크기를 정의합니다.

텍스처 해상도 값은 인치당 픽셀 수(또는 비네팅이 장면 좌표 공간을 지정하는 단위)로 지정됩니다.

재료 해상도는 `res=`으로 지정됩니다.
