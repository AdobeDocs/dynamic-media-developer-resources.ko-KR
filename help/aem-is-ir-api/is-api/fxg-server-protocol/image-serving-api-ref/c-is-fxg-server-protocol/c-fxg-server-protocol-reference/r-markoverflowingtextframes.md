# markFallowedTextFrames{#markoverflowingtextframes}

더하기 기호가 있는 넘치는 텍스트 프레임을 표시합니다. 텍스트 오버플로우 표시기는 텍스트가 텍스트 프레임(또는 스레드 텍스트인 경우 마지막 텍스트 프레임)에서 지정된 공간을 초과하는 경우를 보여 줍니다. 이 표시기는 위에 더하기 기호가 있는 빨간색 상자입니다.

<table id="simpletable_F17FD29EB52043BF9000923ED5195A26"> 
 <tr class="strow"> 
  <td class="stentry"> <p><span class="codeph"> &amp;markFallowedTextFrames</span> </p> </td> 
  <td class="stentry"> <p>0|1 </p></td> 
 </tr> 
</table>

URL 호출을 통해 수정자 `markOverflowingTextFrames=1`을 설정하면 텍스트가 더하기 기호로 겹쳐진 모든 텍스트 프레임이 표시됩니다. 또한 Dynamic Media Classic 미리 보기에서 텍스트 오버세트 표시기는 기본적으로 &quot; `TRUE`&quot;로 설정됩니다.

기본값은 0입니다.
