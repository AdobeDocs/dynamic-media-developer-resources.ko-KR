# effectMask{#effectmask}

요청에 대한 속성::ApplyEffectMask 값을 무시합니다.

effectMask는 글로벌 수정자입니다. 요청의 모든 효과 레이어에 적용됩니다.

유효한 값은 0과 1입니다. 0은 효과 마스크를 적용하지 않음을 의미합니다. 1은 효과 마스크를 적용함을 의미합니다. 기본값은 1입니다.
