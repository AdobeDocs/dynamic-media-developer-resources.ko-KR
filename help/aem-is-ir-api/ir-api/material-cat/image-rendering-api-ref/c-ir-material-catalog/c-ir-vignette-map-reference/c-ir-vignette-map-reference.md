# 비네팅 맵 참조{#vignette-map-reference}

재료 카탈로그에 비네팅 맵이 필요한 경우 비네팅 맵 파일의 상대 또는 절대 경로를 특성::VignetteMapFile에 지정해야 합니다.

특정 자료 카탈로그의 비네팅 맵 파일에 있는 항목은 기본 카탈로그의 비네팅 맵 파일에 있는 항목을 덮어씁니다.
