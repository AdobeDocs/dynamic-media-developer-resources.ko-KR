# setVal{#setval}

s7:elementIDм—җ лҢҖн•ң н…ҚмҠӨнҠё л…ёл“ң к°’мқ„ м„Өм •н•ҳмӢӯмӢңмҳӨ.

`setVal.elementID= *[!DNL value]*`

FXG л…ёл“ң мҡ”мҶҢм—җ `s7:elementID`мқҙ(к°Җ) м •мқҳлҗҳм–ҙ мһҲмңјл©ҙ н•ҙлӢ№ л…ёл“ңмқҳ н…ҚмҠӨнҠё к°’мқ„ мЎ°мһ‘н•  мҲҳ мһҲмҠөлӢҲлӢӨ.
