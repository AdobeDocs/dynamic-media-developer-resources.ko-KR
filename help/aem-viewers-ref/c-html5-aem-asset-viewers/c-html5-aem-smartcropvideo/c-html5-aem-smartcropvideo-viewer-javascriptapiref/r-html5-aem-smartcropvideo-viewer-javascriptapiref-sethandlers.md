# setHandler{#sethandlers}

스마트 자르기 비디오 뷰어에 대한 JavaScript API 참조입니다.

`setHandlers(handlers)`

0개 이상의 콜백 핸들러를 지정합니다. 이 메서드 호출은 해당 뷰어 인스턴스에 대해 이전에 할당된 이벤트 핸들러를 완전히 덮어씁니다. 전에 호출해야 함 `init()`.
