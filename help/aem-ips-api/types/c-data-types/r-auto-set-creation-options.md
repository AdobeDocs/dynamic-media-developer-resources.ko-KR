# AutoSetCreationOptions{#autosetcreationoptions}

업로드 작업을 위한 자동 설정 생성 스크립트 목록. 업로드에 대해 지정된 모든 스크립트가 업로드된 모든 자산에 적용된다고 가정합니다.
