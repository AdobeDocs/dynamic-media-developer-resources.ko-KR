# loadcache{#loadcache}

서버 캐시 미리 로드 req=img와 마찬가지로 요청을 실행하지만, MIME 유형 text/plain의 텍스트 데이터로 형식이 지정된 응답 이미지(image.length)의 길이를 반환합니다.

`req=loadcache`

HTTP 응답에 액세스할 수 없습니다.

요청의 다른 명령은 문서화된 대로 적용됩니다.
