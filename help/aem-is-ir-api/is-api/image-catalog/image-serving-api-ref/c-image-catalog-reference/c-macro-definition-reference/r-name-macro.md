# 이름{#name}

매크로 이름.

매크로 이름은 ASCII 문자, 숫자, &#39;-&#39;, &#39;_&#39; 및 &#39;.&#39;의 조합으로 구성될 수 있습니다.

## 속성 {#section-3952a070c21646ab914cc76f289ed8ce}

텍스트 문자열입니다. 필수. 테이블 내에서 각 매크로::Name 값은 고유해야 합니다. 매크로 이름은 대소문자를 구분하지 않습니다.
