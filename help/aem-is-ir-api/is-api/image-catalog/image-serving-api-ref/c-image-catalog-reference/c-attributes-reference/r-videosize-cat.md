# VideoSize{#videosize}

비디오 크기입니다. 각 비디오 레코드와 연결된 크기를 지정할 수 있습니다.

비디오 크기 데이터를 내보내기 위해 `req=set` 요청에 사용됩니다.

## 속성 {#section-93a2631a57e2423ea6b40f68c55062a1}

너비와 높이에 대해 쉼표로 구분된 정수 값(예: 512,288).

## 기본값 {#section-baaee759ef344c54bbc014dc8944768a}

없음.
