명령 매크로의 이름입니다.

`&$ *`이름`*$`

*`name`* 명령은 대/소문자를 구분하지 않으며 문자, 숫자, `-`(하이픈), `_`(밑줄) 및 `.`(마침표) 문자의 조합으로 구성될 수 있습니다.
