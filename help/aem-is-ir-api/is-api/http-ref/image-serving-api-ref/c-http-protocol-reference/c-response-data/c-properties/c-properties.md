# 속성{#properties}

다음 `req=` 유형에 대한 응답으로 속성 데이터가 반환됩니다.

* `catalogprops`
* `imageprops`
* `props`
* `userdata`

`userdata` 응답은 `catalog::UserData`의 내용이 속성 형식을 따르는 경우에만 속성 형식을 갖습니다.

* [JavaScript 속성](r-javascript-properties.md)
* [XML 속성](r-xml-properties.md)
* [JSONP 속성](r-json-properties.md)
