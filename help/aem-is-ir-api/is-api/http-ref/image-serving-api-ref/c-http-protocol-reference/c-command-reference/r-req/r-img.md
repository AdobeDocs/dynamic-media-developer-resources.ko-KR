# img{#img}

이미지(기본값). 표준 이미지 데이터를 요청합니다.

`req=img`

응답 데이터 형식 및 응답 MIME 형식은 `fmt=`에 의해 결정됩니다. `req=img` 한정자는 기본 요청 형식이므로 명시적으로 필요하지 않습니다. `catalog::Expiration`을(를) 기반으로 하는 TTL로 HTTP 응답을 캐시할 수 있습니다.

다른 요청 명령은 문서화된 대로 적용됩니다.
