# PageView.enableHD{#pageview-enablehd}

` [PageView.|<containerId>_pageView.]enableHD=always|never|limit[, *`수`*]`

## 속성 {#section-38b878e38caa439bb462d4fa637afdaa}

선택 사항입니다.

## 기본값 {#section-50b22de303c1432094530e6583132c02}

`limit,1500`
