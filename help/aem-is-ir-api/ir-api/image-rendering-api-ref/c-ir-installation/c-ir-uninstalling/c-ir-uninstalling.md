# 이미지 렌더링 제거 중{#uninstalling-image-rendering}

필요한 경우 이미지 렌더링을 제거할 수 있습니다.

설치 폴더에서 일부 사용자 문서를 제거할 수 있습니다. 제거하기 전에 유지할 모든 파일을 백업합니다.

제거하기 전에 관리 또는 루트 권한으로 서버 호스트에 로그인합니다.
