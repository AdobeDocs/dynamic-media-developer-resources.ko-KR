# 반환{#returns}

vntc는 처리 중 오류 발생 여부에 따라 다음 값을 반환합니다.

* 오류가 발생하지 않으면 0입니다.
* 오류(치명적 또는 치명적이지 않음)가 발생한 경우 1.
