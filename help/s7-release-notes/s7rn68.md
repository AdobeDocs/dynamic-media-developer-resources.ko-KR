## 릴리스 특징 {#section-524cffc79c1f4bb38627a5ecccd0bf52}

* 응답형 웹 디자인에 대한 지원이 추가됨—페이지 레이아웃에 따라 동적으로 크기를 조정하는 중단점 및 응답형 HTML5 뷰어(이미지 세트, 비디오, 혼합 미디어, eCatalog, Spinsets, Switchset, Zoom)를 기반으로 하는 응답형 이미지 제공.
* Adobe Analytics 비디오 보고서 통합.
* 고급 HTML5 뷰어 사용자 지정— 포함 웹 페이지를 사용하여 사용자 지정 뷰어 상호 작용을 만듭니다.
* 이미지 개선 사항 - 게재 시 포함된 경로에 이미지를 자르는 기능이 추가되었습니다.

## 뷰어 {#section-609885a9c8514a4095da74a0b526138d}

다음 사항이 업데이트되었습니다.

* Windows 운영 체제에 대한 최소 시스템 요구 사항은 이제 Windows 7입니다.
* 이제 Internet Explorer에 대한 최소 시스템 요구 사항은 버전 9입니다.
* macOS에 대한 최소 시스템 요구 사항은 이제 macOS 10.8입니다.
* Safari에 대한 최소 시스템 요구 사항은 이제 버전 6입니다.

리팩터링된 항목:

* 반응형 디자인을 지원하는 HTML5 뷰어.
* CSS 입력 및 크기 마커를 사용하여 데스크톱 및 모바일 장치에서 표시를 제어하는 HTML5 뷰어.

이제 다음 항목이 지원됩니다.

* HTML5 비디오 뷰어와 혼합 미디어 뷰어에 대한 Adobe Analytics 비디오 보고서 통합.
* 동일한 페이지에서 여러 HTML5 뷰어에 대한 이벤트 추적.
* 추가됨 `SetIndicator` HTML5 [확대/축소 뷰어] 및 [HTML5 혼합 미디어 뷰어]를 사용하여 모바일 장치에서 색상 견본을 시각적으로 나타낼 수 있습니다.
* 추가됨 `getComponent` HTML5 뷰어 구성 요소에 대한 외부 액세스를 허용하는 API.
* 단일 JSON 개체로 전달된 HTML5 뷰어 구성 매개 변수.
* WebVTT 캡션을 JSON 개체로 전달하여 규칙 세트를 구성할 필요가 없습니다.
* 이미지 맵을 외부 페이지에 연결하기 위한 HTML5 eCatalog 뷰어의 JavaScript 템플릿입니다.
* HTML 5 eCatalog 뷰어 내의 페이지에 연결.
* `stagesize` HTML5 플라이아웃 뷰어로 이동합니다.
* `initComplete` 뷰어 후 함수를 트리거하기 위한 HTML5 뷰어에 대한 핸들러.
* 타사 추적을 지원하기 위한 HTML5 뷰어에 대한 Analytics 핸들러입니다.
* 소셜 공유를 위해 링크 및 포함 코드를 쉽게 가져올 수 있도록 &quot;모두 선택&quot; 단추가 추가되었습니다.
* FlyoutZoomView에 대한 크기 조정이 추가되었습니다.
* 응답형 디자인에서 사용할 때 FlyoutZoomView에 대한 이미지 다시 로드 중단점 설정
* 볼륨 조정을 위한 클릭 지원이 추가되었습니다.
* SpinView 및 ZoomView를 사용하여 뷰어와 상호 작용하지 않을 때 웹 페이지를 포함하는 데 제스처를 적용할 수 있습니다.
* 추가됨 `unload()` SpinView 및 ZoomView에 대한 API를 사용하여 런타임 시 구성 요소를 배치할 수 있습니다.
* 뷰어와 상호 작용하지 않을 때 포함 웹 페이지에 제스처를 적용하는 비디오 뷰어 .
* ZoomView에 대한 초기 확대/축소 영역을 설정합니다.
* 숨기려면 확대/축소 보기 `IconEffect` 확대/축소가 불가능한 구성 요소

참조 - [뷰어 참조 안내서](https://experienceleague.adobe.com/docs/dynamic-media-developer-resources/library/homeviewers.html?lang=en).

## 버그 수정 {#section-456515477c8a4aa69309de43292ce5ed}

**뷰어**

* 잠재적 Flash Fastzoom 및 Flyout 뷰어 보안 문제입니다.
* 인식할 수 없는 UserAgent로 인해 HTML5 뷰어가 로드되지 않았습니다.
* HTML5 eCatalog 뷰어 마우스 드래그 앤 드롭이 Internet Explorer에서 이미지 맵이 포함된 페이지에서 작동하지 않습니다.
* HTML5 비디오 뷰어는 Flash 플러그인을 차단 해제할 수 있는 액세스를 차단합니다.
* HTML5 뷰어는 Firefox 브라우저에서 인쇄되지 않습니다.
* HTML5 확대/축소 뷰어에 확대/축소를 시작하거나 재설정할 때 약간 지터가 표시됩니다.
* HTML5 및 AS3 뷰어에 대한 HTTPS 지원.
* autoResize는 HTML 플라이아웃 뷰어 복사 URL/포함 코드에 있지 않아야 합니다.
* HTML5 비디오 뷰어 복사 URL에 videoserverurl 수정자가 없습니다.
* 매개 변수 `contenturl` 게시 서버 HTML5 뷰어 미리 보기 URL에서 가져옵니다.

**이미징**

* 이미지 게시 설정에서 조정된 보기를 기본 이미지로 추가할 수 없습니다.
* colorManagementOptions 레이블 &quot;SRGB로 변환(기본값)&quot;을 &quot;기본 색상 관리&quot;로 변경합니다.
* 파일 > 정보 편집 > 여러 이미지 자산을 선택하는 경우, 정보 편집을 사용할 수 없었습니다.

**플랫폼**

* Adobe Analytics 구성을 편집한 다음 저장하지 않고 설정에서 이스케이프 처리를 시도할 때 경고 메시지가 추가되었습니다.
* 예약된 업로드 작업을 편집할 때 프로젝트 옵션 참조가 없음으로 재설정됩니다.
* 사용자 역할에 대해 파일 이동 폴더가 표시되지 않아야 합니다.
* 세부 사항 보기에서 일부 탭을 볼 수 없습니다.
* Flashplayer 디버그 버전이 설치된 경우 Web-to-Print 편집 오류가 발생합니다.
* Space After가 작동하지 않습니다. 사용해야 합니다. `sa`, 현재 `margb`.
* 회사 설정을 초기화할 때 Scene7 Publishing System에서 중복 옵션을 보냅니다 `<defaultValue>` API에서 `<createMetadataField>`.

* Flash 메모리가 부족하여 Flash 플러그인 및 Scene7 게시 시스템이 충돌했습니다.
* 날짜/시간 사용자 정의 필드의 값을 제거할 때 Scene7 Publishing System은 IPS(Image Publishing System)에 요청을 보내지 않습니다.
* 비디오 인코딩 옵션이 활성화되어 있으면 PDF 또는 PSD 파일을 재처리하면 응용 비디오 인코딩 경고 메시지가 표시됩니다.
* SSL이 활성화되어 있을 때 자산을 내보내지 못했습니다.
* 암호 분실 전자 메일에 잘못된 암호가 표시되었습니다.
* 모든 Scene7 Publishing System 웹 관리 요청은 HTTP로 리디렉션됩니다.
* 사전 설정을 선택하지 않은 경우 업로드 중에 모든 일괄 세트 사전 설정이 적용되었습니다.
* s7info 콘텐츠를 업로드할 때 HTTP를 통해 Scene7 Publishing System 요청 s7info.
* 암호 분실 이메일의 URL은 SSL Scene7 Publishing System에서 온 경우에도 HTTP입니다.
* 특수 문자가 &quot;암호를 잊다&quot; 전자 메일 형식의 HTML으로 인코딩되어 암호가 올바르게 표시되지 않았습니다.
* 평가판 계정에 대한 시작 이메일에 Scene7 FTP 계정 정보가 누락되었습니다
* 프로젝트 옵션과 자르기 옵션을 모두 선택한 경우 작업을 다시 처리하지 못했습니다.
* 작업 옵션이 회사 기본값으로 재설정되면 업로드 파일이 작동하지 않습니다.
* Company 레이블이 Adobe Analytics 설정 페이지에 완전히 표시되지 않습니다.
* 메타데이터를 가져올 수 없습니다.

## 알려진 문제 {#section-b2608177928b4174b080d89c938bb0d3}

* Adobe Illustrator 및 Adobe InDesign의 2014년 5월 업데이트에서는 Scene7 FXG를 만드는 플러그인과의 호환되지 않는 기능이 도입될 예정입니다. 이러한 비호환성은 Scene7 Web-to-Print 솔루션 사용자에게 영향을 줍니다. Scene7 Web-to-Print를 사용하는 경우 Illustrator 및 InDesign을 이 Creative Cloud 업데이트으로 업그레이드하지 마십시오. 대신 2014년 5월 이전 버전을 계속 사용하십시오. Adobe Inc은 2014년 가을 Illustrator 및 InDesign의 최신 버전과 Scene7 FXG 작성의 호환성을 복원할 예정입니다.
* 범용 뷰어 사전 설정에 비활성화된 기본 AS2 사전 설정이 포함되어 있으면 포함 코드 단추가 작동하지 않습니다.
* 응용 비디오 세트를 재처리하면 항상 새 응용 비디오 세트가 만들어집니다.
* 작업 세부 정보 대화 상자의 사용자 인터페이스는 세부 정보에 시간을 추가한 후 오른쪽에 잘립니다.

## 더 이상 지원되지 않는 기능 {#section-7b04b6fdb8d1473880ccaf84211254ad}

* Scene7 6.9 가을 릴리스에서 텍스트 하이픈 넣기가 지원되지 않습니다. 하이픈 넣기 사전은 최적의 텍스트 흐름을 지원하기 위해 단어를 분할하고 하이픈을 넣는 방법을 정의합니다.
* 2014년 1월 31일부터 Scene7은 공식적으로 DHTML 뷰어 플랫폼에 대한 지원을 종료했습니다.
* 동적(Flash) 배너는 이 릴리스에서 더 이상 사용되지 않습니다. 개인화된 웹 사이트 배너를 만들고, 관리하고 제공하는 다른 기술이 등장함에 따라 이 기능의 채택은 거의 이루어지지 않았습니다. Scene7 동적(Flash) 배너 기능은 서버측에서 계속 작동하지만 더 이상 지원되지 않습니다.
* 미리 알림: 메타데이터 서버 및 관련 API의 사용은 이전 Scene7 릴리스부터 공식적으로 지원되지 않습니다.
* Scene7 Creative Suite 확장 프로그램(Flash 패널)은 2014년 말 더 이상 사용되지 않습니다. 콘텐츠 수집 워크플로우의 추가 개발 작업은 Adobe Experience Manager Dynamic Media 및 Creative Cloud에 집중됩니다. Scene7의 경우 Scene7 Publishing System의 업로드 및 내보내기 기능은 그대로 유지되며 대신 사용할 수 있습니다.
* 2014년 5월 현재 Akamai는 Scene7의 캐시 무효화가 완료되면 더 이상 이메일 알림을 지원하지 않습니다. CDN 무효화에 대한 일반적인 처리 시간은 약 8분입니다. 요청 대기열이 많거나 URL이 많은 경우 처리 시간이 약간 더 걸릴 수 있습니다.
* 세부 사항 보기의 URL 및 포함 코드 탭에 있는 &quot;범용 사전 설정&quot; 카테고리입니다.
* 비디오 세부 사항 보기를 위한 RTMP 스트리밍.
* Scene7 Publishing System의 &quot;이메일 통신&quot; 및 SendMailSimple.jsp 서블릿.
* Media Portal 사용자에 대한 이메일 템플릿 시작 페이지의 Scene7 FTP 계정 섹션.
* IPS(Image Publishing System) 인스턴스가 다운되어 필터 패널의 dc.title.
* HTML5 뷰어에서 전체 화면 모드로 표시되는 소셜 기능입니다.

## Scene7 SDK 2.7.1의 새로운 기능 및 버그 수정 {#section-b82b4f546ce44a769e94fb1493511035}

**일반**

* iOS6에 대한 최소 요구 사항이 증가했습니다.
* 고정: Safari 도구 설명이 전체 화면 모드에서 화면 밖으로 이동되었습니다.
* 다양한 버그를 해결하기 위한 도구 설명을 리팩터링했습니다.

**단추**

* 기본 단추 스타일에 CSS에 대한 지원을 추가했습니다.

**컨테이너**

* 고정: 전체 화면에서 반환하고 자산을 교체한 후 잘못된 컨테이너 크기입니다.
* 고정: 컨테이너가 CSS 변형으로 변경되면 컨테이너 좌표가 올바르지 않습니다.

**EmailShare**

* 고정: 제거 단추에서 발생하는 도구 설명 아티팩트.
* 고정: 스크롤 막대 도구 설명은 사용자 인터페이스 뒤에 표시되며 잘립니다.
* 고정: 전자 메일 보내기 단추 뒤에 나타나는 &quot;취소&quot; 도구 팁입니다.

**EmbedShare**

* 고정: 포함 코드 텍스트가 줄바꿈되지 않고 Internet Explorer의 스크롤 막대 위에 표시됩니다.

**FlyoutZoomView**

* 고정: 플라이아웃 이벤트는 Internet Explorer 8에서 발송되지 않습니다.

**가변 볼륨**

* 고정: 볼륨 막대 아래에 표시되는 볼륨 도구 팁입니다.

**견본**

* 고정: `selectSwatch` 에 포함할 때 트리거하지 않는 메서드 `initViewer()` 함수 위에 있어야 합니다.

**SocialShare**

* 공유 단추의 레이아웃 위치를 제어하기 위해 베어링 수정자에 대한 지원을 추가했습니다.
* 소셜 기능과 상호 작용할 때 일시 중지/재생을 제어하는 이벤트가 추가되었습니다.
* 고정: 소셜 공유 아이콘이 `controlbar` 확대/축소.
* 고정: Galaxy SIII에서 처음 페이드 인할 때 소셜 공유 아이콘이 표시되지 않습니다.
* 고정: 소셜 공유 모달 대화 상자가 iPad 3에서 전체 화면 모드로 표시되지 않습니다.
* 고정: Safari 5.1을 사용하는 Mac에서 소셜 공유 모달 대화 상자가 전체 화면 모드로 표시되지 않습니다.
* 고정: 소셜 공유 양식 대화 상자가 스크롤한 후 화면에 표시되지 않습니다.

**ThumbnailGridView**

* 고정: ThumbnailGridView가 전송하지 않음 `AssetEvent.ASSET_CHANGED`.

**VideoPlayer**

* 에 대한 지원이 추가되었습니다 `initialBitRate` 수정자 - 데스크탑에서 초기 재생에 사용되는 비트율을 선택합니다.
* iPhone 장치의 비디오 문제를 해결하기 위해 iPhone 비디오 프록시를 구현했습니다.
* 소셜 기능이 호출될 때 재생을 일시 중지하는 이벤트 리스너에 대한 지원이 추가되었습니다.
* 이제 자동으로 설정된 재생 모드가 Safari 데스크탑에서 기본 플레이어를 사용합니다.
* 고정: Flash 재생을 사용하여 IE8에서 보안 경고를 생성하는 HTTPS 프로토콜입니다.
* 고정: Flash 재생을 사용하여 Safari 5의 전체 화면 모드에서 비디오 크기가 올바르게 지정되지 않습니다.
* 고정: iPhone에서 비디오가 재생되면 미디어 세트의 색상 견본을 선택할 수 없습니다.

**확대/축소 보기**

* 고정: Android™에서 이미지로 확대하거나 회전하는 장치로 확대하면 표시되는 아티팩트입니다.
