# Text string{#text-string}

텍스트 문자열 데이터 유형입니다.

문자열에 `<CR>`일부, `<LF>`또는 `<TAB>` 문자가 포함되어 있지 않은 경우 문자열 구분 기호는 선택 사항입니다. 단일 또는 큰따옴표는 구분 기호로 사용할 수 있습니다. 인용 부호가 사용되는 경우, 문자열 내에 포함된 이러한 인용 부호는 연속된 두 개의 따옴표(예:이번 달 특별 행사).
