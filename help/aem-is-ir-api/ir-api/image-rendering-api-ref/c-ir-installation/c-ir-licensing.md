# 라이선스{#licensing}

Dynamic Media 이미지 렌더링 라이센스 키가 필요하며 이미지 렌더링 이 설치된 각 시스템에 설치해야 합니다.

유효한 라이센스가 없으면 이미지 렌더링이 실행되지 않습니다. 설치 절차에는 라이센스 키를 가져오고 설치하는 단계가 포함됩니다.
