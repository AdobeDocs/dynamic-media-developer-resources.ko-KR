# 자료 데이터 참조{#material-data-reference}

재료 카탈로그에 재료 데이터가 필요한 경우 재료 데이터 파일의 상대 경로나 절대 경로를 attribute::CatalogFile에 지정해야 합니다.

기본 카탈로그의 자재 데이터 파일에 있는 특정 자재 카탈로그 대체 항목의 자재 데이터 파일 항목.
