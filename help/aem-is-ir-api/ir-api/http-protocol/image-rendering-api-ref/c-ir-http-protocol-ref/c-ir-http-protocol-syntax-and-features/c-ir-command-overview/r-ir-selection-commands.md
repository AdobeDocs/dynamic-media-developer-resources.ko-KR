# 선택 명령{#selection-commands}

이러한 명령은 비네팅 그룹, 객체, 그룹 또는 객체의 하위 영역을 선택하는 데 사용됩니다.

명령 또는 재료 또는 둘 다 이 선택 명령 뒤에, 다음 선택 명령(또는 요청의 끝)이 선택한 항목에 적용되기 전에 지정됩니다. 선택 명령은 재료 사양 세그먼트(MSS)를 구분합니다.
