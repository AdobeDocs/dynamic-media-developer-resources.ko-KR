# 명명된 레이어{#named-layers}

layer= 명령을 사용하면 레이어에 이름을 할당하고 나중에 레이어 번호가 아닌 이름으로 동일한 레이어를 참조할 수 있습니다.

이렇게 하면 템플릿에서 변수를 사용하지 않고도 요청 URL에서 참조할 수 있는 명명된 레이어를 정의할 수 있습니다.
