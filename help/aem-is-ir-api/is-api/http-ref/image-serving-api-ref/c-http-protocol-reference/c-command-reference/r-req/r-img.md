# img{#img}

이미지(기본값). 표준 이미지 데이터를 요청합니다.

`req=img`

응답 데이터 형식 및 응답 MIME 유형은 `fmt=`으로 결정됩니다. `req=img` 은 기본 요청 유형이며 명시적으로 지정할 필요가 없습니다. HTTP 응답은 `catalog::Expiration`을(를) 기준으로 TTL을 사용하여 액세스할 수 있습니다.

다른 요청 명령은 문서로 적용됩니다.
