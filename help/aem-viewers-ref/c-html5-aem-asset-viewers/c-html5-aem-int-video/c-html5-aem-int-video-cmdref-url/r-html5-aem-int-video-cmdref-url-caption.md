# caption{#caption}

대화형 비디오 뷰어용 URL 명령.

` caption= *`파일`*[,0|1]`

뷰어는 호스팅되는 WebVTT 파일을 통한 자막을 지원합니다. 겹치는 큐 및 영역은 지원되지 않습니다. 지원되는 큐 위치 지정 연산자는 다음과 같습니다.
