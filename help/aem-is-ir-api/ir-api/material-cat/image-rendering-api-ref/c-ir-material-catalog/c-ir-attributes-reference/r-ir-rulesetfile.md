## 속성 {#section-e805898763824caba24abb6af1706f5c}

## 기본값 {#section-69b1f865aa8d476bb087dc862e903ab1}

비어 있음 - 이 재질 카탈로그에 개인 요청 사전 처리 규칙이 없음을 나타냅니다.
