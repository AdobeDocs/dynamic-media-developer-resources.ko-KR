# 키보드 액세스 가능성 및 탐색{#keyboard-accessibility-and-navigation}

뷰어 UI에 노출된 모든 기능은 키보드로 액세스할 수 있습니다.
최종 사용자는 `Left`, `Right`, `UP` 및 `DOWN` 키를 눌러 이미지를 패닝합니다.
다음 `Esc` 키 입력을 사용하여 이미지를 재설정할 수 있습니다.
