# setVideo{#setvideo}

Video360 뷰어용 JavaScript API 참조

`setVideo(videoUrl)`

새 외부 비디오를 설정합니다. `init()` 전후에 언제든지 호출할 수 있습니다. `init()` 이후에 호출되는 경우 뷰어는 런타임에 비디오를 교환합니다.
