필수. `asset` 매개 변수를 사용하지 않는 경우. [외부 비디오 지원](../../../c-html5-s7-aem-asset-viewers/c-html5-video-reference/r-html5-video-viewer-20-external-video-support.md#concept-22c67fee43274a29b28ee16770b1b1f3)을 참조하세요.)
