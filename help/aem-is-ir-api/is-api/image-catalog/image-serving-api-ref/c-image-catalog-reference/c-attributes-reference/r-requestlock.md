# RequestLock{#requestlock}

잠금 모드 요청. 요청에 포함되어야 하는 요청 잠금 유형을 지정합니다.

## 속성 {#section-ae89274d3b7940d0bf3d816f7c2f51fd}

플래그. 요청 잠금을 비활성화하려면 0으로 설정하고 간단한 해시 잠금을 선택하려면 1로 설정합니다.

## 기본값 {#section-fd5b5f0417bd444189b20494d179193e}

상속 위치 `default::RequestLock` 정의되지 않은 경우 또는 비어 있는 경우.
