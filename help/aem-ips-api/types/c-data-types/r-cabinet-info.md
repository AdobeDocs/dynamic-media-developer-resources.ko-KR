| 이름 | 유형 | 설명 |
|---|---|---|
| `*`originalPath`*` | `xsd:string` | 원래 캐비닛의 상대 경로입니다. |

| `*`fileSize`*` | `xsd:int` | 캐비닛 크기. |
| `*`imagePath`*` | `xsd:string` | 비네팅과 연결된 캐비닛의 상대 경로입니다. |

| `*`optimizedFile`*` | `xsd:string` | IPS에 최적화된 캐비닛 파일. |
