# Scene7 Publishing System 6.0 릴리스 노트{#scene-publishing-system-release-notes}

Adobe Marketing Cloud의 일부분인 Adobe Scene7 Publishing System 6.0의 기능, 개선 사항 및 수정 사항을 설명하는 릴리스 노트입니다.

**일반**

* 중국어 지원

   [이미지 사전 설정을 기반으로 이미지 자산 미리 보기](http://help.adobe.com/en_US/scene7/using/WS98ca2e6790647c06-2ce305113564963202-7ffe.html)

   이 [!DNL Embed Code] 기능은 다음 자산 유형에 대해 [!DNL Preview: Viewer List] 페이지 또는 [!DNL URLs] 패널을 열 때 사용할 수 [!DNL Detail View] 있습니다.

**Media Portal**

* 대용량 파일을 시간 초과하지 않고 내보낼 수 있습니다.
* 작업 그룹(관리자)에서 사용자를 위한 FTP 계정을 만듭니다.
* 그룹 및 자산/권한(관리자)이 있는 사용자 목록을 내보냅니다.

* 메타데이터 스키마

   메타데이터 스키마는 메타데이터 속성 집합을 정의합니다. 메타데이터 스키마의 각 속성에는 문자열, 부동, 정수, 날짜 또는 예/아니요와 같은 기본 데이터 유형이 있습니다. 목록, 열린 선택 항목 또는 닫힌 선택 항목을 정의할 수도 있습니다. 검색, 편집, 가져오기, 메타데이터 보기 및 메타데이터 사전 설정과 같은 모든 메타데이터 워크플로우에서 메타데이터 스키마를 사용할 수 있습니다. [추가 정보](http://help.adobe.com/en_US/scene7/using/WS259993e42159a215-1c6a66df1265272619e-7fec.html#WSd968ca97bf03cf72-5e3dd3a113268dc80f5-8000).

   메타데이터 스키마는 UDF에 대한 후속 기술로, 특히 메타데이터 서버 컨텍스트에서 사용자 지정된 메타데이터 워크플로를 위해 사용됩니다.

* 메타데이터 서버

   >[!NOTE]
   >
   >메타데이터 서버는 5월부터 사용할 수 있습니다. 지원 기능으로 활성화되어야 합니다. 메타데이터 서버는 현재 북미와 유럽에서만 사용할 수 있습니다.

* 메타데이터 서버는 자산의 모든 메타데이터 필드 하위 집합을 공개 서버에 노출하며, eCom 페이지 또는 서버에 의해 여기에서 요청할 수 있습니다. 이렇게 노출된 메타데이터는 자산 속성, UDF, XMP 및 메타데이터 스키마 속성의 혼합 집합일 수 있습니다. 메타데이터 서버는 메타데이터 또는 이미지 유사성 또는 두 가지 모두 검색과 같은 다양한 검색 기능이 있는 WEB-API를 제공합니다. 반환된 각 자산에 대해 메타데이터 속성 목록이 필요합니다. [추가 정보](http://help.adobe.com/en_US/scene7/using/WS1ffbb36e209a6fc9-44279087131d3ad5622-8000.html).
* 메타데이터 가져오기를 통해 XMP 속성 처리

   메타데이터 가져오기가 메타데이터 스키마에 정의된 XMP 속성 및 속성을 처리하기 위해 확장되었습니다.
* 값이 포함된 필드만 표시하려면 메타데이터 보기

   값이 포함된 메타 속성만 표시하는 &quot;값이 있는 모든 속성&quot;이라는 새로운 사전 정의된 메타데이터 보기가 있습니다. 값이 없는 속성(XMP, 사용자 정의 필드 또는 새 메타데이터 스키마)은 숨겨지고 표시되지 않습니다.
* 이제 메타데이터 설정이 관리 사용자 인터페이스로 그룹화됩니다.

   이제 **[!UICONTROL 설정 > 애플리케이션]**&#x200B;설정에서 메타데이터 관련 설정 페이지가 최상위 섹션인 최상위 섹션으로 그룹화됩니다 [!DNL Metadata]. 메타데이터 보기, 메타데이터 사전 설정, 메타데이터 스키마 및 사용자 정의 필드에 대한 설정 페이지를 확인할 수 있습니다.

이 데스크탑 애플리케이션을 사용하면 기존 3D 모델을 가져와 2D 비네팅 파일로 작성할 수 있습니다. 결과 비네팅 파일은 SPS로 업로드하여 필요할 때 색상과 텍스처를 바꿀 수 있습니다.

## 소프트웨어 수정 사항 {#section-4219d5dd0a024cd3a95aaac45598363c}

메타데이터 서버에서 다음 문제가 해결되었습니다.

* 매우 큰 EPS 파일 업로드가 실패하는 문제를 해결했습니다.
* 긴 결과 목록이 있는 &quot;메타데이터 검색&quot;은 자동으로 실패합니다. 이 문제로 인해 잘못된 검색 결과가 표시됩니다.
