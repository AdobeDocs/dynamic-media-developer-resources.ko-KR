# 효과 마스크{#effectmask}

요청에 대한 attribute::ApplyEffectMask 값을 재정의합니다.

effectMask는 전역 한정자입니다. 이 설정은 요청의 모든 효과 레이어에 적용됩니다.

유효한 값은 0과 1입니다. 영(0)은 효과 마스크를 적용하지 않음을 의미합니다. 하나(1)는 효과 마스크를 적용하는 것을 의미한다. 기본값은 1입니다.
