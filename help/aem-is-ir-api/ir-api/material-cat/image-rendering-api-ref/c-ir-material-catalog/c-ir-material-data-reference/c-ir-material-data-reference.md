# 자료 데이터 참조{#material-data-reference}

재료 카탈로그에 재료 데이터가 필요한 경우 재료 데이터 파일의 상대 또는 절대 경로를 attribute::CatalogFile에 지정해야 합니다.

특정 재료 카탈로그의 재료 데이터 파일에 있는 항목은 기본 카탈로그의 재료 데이터 파일에 있는 항목을 덮어씁니다.
