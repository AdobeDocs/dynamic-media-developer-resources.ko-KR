# 데이터 캐싱{#data-caching}

이미지 렌더링은 이미지 제공 캐싱 기능을 사용하여 응답 이미지 데이터, 중첩된/포함된 요청에 대한 중간 데이터, 외부 소스에서 가져온 소스 이미지 데이터(텍스처 등)를 저장합니다.

캐싱 메커니즘에 대한 설명은 이미지 제공 설명서를 참조하십시오.
