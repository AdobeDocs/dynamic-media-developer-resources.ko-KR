# AssetType{#assettype}

자산 유형. 카탈로그에 게시된 집합 유형을 식별하는 데 사용됩니다::ImageSet.

자산 유형은 `req=set` 요청에 대해 생성할 응답 유형을 결정합니다. 값을 지정하지 않으면 자동 감지 규칙이 `req=set` 응답 유형을 결정합니다.

## 속성 {#properties}

다음 집합의 열거형 값:

미디어셋

SPINSET

SPINSET2D

VIDEOSET

이미지 집합

ECATALOG

없음.
