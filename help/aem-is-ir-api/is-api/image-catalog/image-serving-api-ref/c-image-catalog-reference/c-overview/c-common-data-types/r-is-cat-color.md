# 색상{#color}

색상 데이터 유형입니다.

색상 값. 16진수로 채워진 RGB 값, 선택적으로 `0x` 앞에 옵니다. 예를 들어 RGB 값 `128,255,0`은 `0x80ff00` 또는 `80ff00` 로 지정할 수 있습니다.
